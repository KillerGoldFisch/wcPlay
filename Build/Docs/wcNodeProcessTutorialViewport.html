Class: wcNodeProcessTutorialViewport

wcNodeProcessTutorialViewport

This node demonstrates an example of using the node's viewport for displaying graphics directly on your node. It can also receive mouse events for interactivity. When inheriting, make sure to include 'this._super(parent, pos);' at the top of your init function.

new wcNodeProcessTutorialViewport(parent, pos)

Parameters:
Name Type Description
parent String The parent object of this node.
pos wcPlay~Coordinates The position of this node in the visual editor.

Event that is called when it is time to draw the contents of your custom viewport. It is up to you to stay within the wcNode.viewportSize you've specified.
