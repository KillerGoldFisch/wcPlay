Class: wcNodeEntry

wcNodeEntry

The base class for all entry nodes. These are nodes that start script chains.
When inheriting, make sure to include 'this._super(parent, pos, name);' at the top of your init function.

new wcNodeEntry(parent, pos, name)

Should be inherited and never constructed directly.

Parameters:
Name Type Argument Default Description
parent String The parent object of this node.
pos wcPlay~Coordinates The position of this node in the visual editor.
name String <optional>
