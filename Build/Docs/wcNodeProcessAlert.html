Class: wcNodeProcessAlert

wcNodeProcessAlert

For debugging purposes, will popup an alert box with a message the moment it is activated. Silent mode will silence this node.
When inheriting, make sure to include 'this._super(parent, pos);' at the top of your init function.

new wcNodeProcessAlert(parent, pos)
