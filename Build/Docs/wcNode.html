Class: wcNode

wcNode

The foundation class for all nodes.
When inheriting, make sure to include 'this._super(parent, pos, name);' at the top of your init functions.

new wcNode(parent, pos, name)

Should be inherited and never constructed directly.

Parameters:
Name Type Argument Default Description
parent String The parent object of this node.
pos wcPlay~Coordinates The position of this node in the visual editor.
name String <optional>
 "Node" The name of the node, as displayed on the title bar.
